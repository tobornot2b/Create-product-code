대리점 상품코드 생성 프로그램
===

##### 업데이트 내역

- 2022.11.15 사이즈 입히는 부분 추가
    - 사이즈코드 -> 복종
    - 사이즈패턴코드 -> 사이즈명


- 프로그램 개요
    1. 사용자에게 몇 가지 조건을 입력받고 시스템 기본정보를 받아와서 상품코드를 조립한다.
    1. 생성되는 엑셀파일 구조
        - 대리점코드 -> 파일명
        - 학교코드 -> 시트명
        - 해당학교로 파생되는 상품코드 기록
        - 해당 대리점에서 사용하는 공통학교 코드도 시트로 생성한다.
    1. 이 프로그램에서 누락된 내용 : 해당대리점의 실제학교 코드가 어떤 공통코드와 매칭되는가? (영업팀에 문의)
